Vortex Merge 개인정보처리방침

1. 수집하는 개인정보

Vortex Merge는 현재 버전에서 사용자의 개인정보를 직접 수집하지 않습니다. 앱은 계정 생성, 로그인, 서버 저장, 채팅, 사용자 생성 콘텐츠 기능을 제공하지 않습니다.

2. 자동으로 수집되는 정보

현재 버전에는 광고 SDK, 분석 SDK, 소셜 로그인 SDK, 위치 기반 기능이 포함되어 있지 않습니다. 따라서 앱 운영자가 광고 식별자, 기기 식별자, 위치 정보, 사용 기록을 별도로 수집하지 않습니다.

다만 앱이 Google Play를 통해 배포되는 경우, Google Play 또는 기기 운영체제는 설치, 업데이트, 비정상 종료, 보안 관련 정보를 자체 정책에 따라 처리할 수 있습니다. 이는 앱 개발자가 직접 수집하는 정보가 아닙니다.

3. 개인정보의 이용 목적

현재 수집하는 개인정보가 없으므로, 개인정보를 이용한 맞춤형 광고, 프로파일링, 마케팅, 제3자 제공을 하지 않습니다.

4. 개인정보의 공유

Vortex Merge는 사용자의 개인정보를 제3자에게 판매, 임대, 공유하지 않습니다.

5. 아동의 개인정보

Vortex Merge는 만 13세 미만 아동을 대상으로 개인정보를 수집하지 않습니다. 보호자가 아동의 개인정보 제공이 있었다고 판단하는 경우 아래 연락처로 문의해 주세요.

6. 데이터 보관 및 삭제

앱이 직접 수집·저장하는 개인정보가 없으므로 별도의 개인정보 보관 기간이 없습니다. 기기 내 게임 진행 정보나 설정이 저장되는 경우 사용자가 앱 데이터 삭제 또는 앱 삭제를 통해 제거할 수 있습니다.
